⚠️ 合規提醒：本工具不會自動送出留言，只幫你找目標貼文 + 產生回覆草稿。請你親自登入 Threads 貼上。避免一天內在大量陌生貼文留明顯業配，那是 Meta 判定 inauthentic behavior 的主要訊號。留言要真心回應對方內容，初殿鍋物的提及要自然、不是每一則都要提。

🔎關鍵字海巡

點關鍵字 → 新分頁開 Threads 搜尋頁（依「最新」排序）。找到值得回的貼文，複製連結和內文，貼到右邊的產生器。

重置成預設清單

會清空你目前的關鍵字，換成內建的火鍋 / 聚餐 / 台中美食 等預設組合。

💬留言靈感產生器

把對方貼文內容貼進來，選風格，按「產生」。模板版不用 API、變化少；Gemini 版會針對貼文量身回覆，自然很多。

朋友口吻 好奇詢問 感同身受 吃貨共鳴
檢查 Gemini 代理設定中…

✅已留言記錄

避免同一則貼文重複留言（很容易被鎖）。每次按「已貼」會把連結加到這裡。

📸UGC 蒐集（顧客提到初殿鍋物的貼文）

看到有人 tag 或提到初殿鍋物，貼連結進來存檔。方便之後聯絡合作或轉發。

⚙️ 進階設定（Apps Script 代理網址）

部署步驟：Apps Script → 部署 → 新增部署 → 類型「Web 應用程式」 → 執行身分「我」 → 存取權「所有人」 → 部署 → 複製網址貼這裡。
金鑰：左側齒輪「專案設定」→ 指令碼屬性 → 新增 GEMINI_API_KEY = 你的 AIzaSy... 金鑰。
程式碼在 threads海巡-script.gs（整份貼到 Apps Script 編輯器）。
